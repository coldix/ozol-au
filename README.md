# OZ On Line (ozol.au)

This repository contains the historical timeline and portfolio for Colin Dixon's 40+ year journey in Australian tech, hosted at `ozol.au`.

## 📁 Project Structure

- `index.html` - Main landing page containing the interactive timeline.
- `css/style.css` - Custom styling utilizing a scalable, dark/light theme glassmorphism design.
- `js/script.js` - Minimal scripts to handle UI logic, theme toggles, and modal form actions.
- `robots.txt` & `sitemap.xml` - Standard SEO files.
- `images/` - Contains gallery and hero images.

## Notes
- Theme states (light/dark) are saved locally.
- Form submissions generate mailto links directly instead of relying on a backend.
- Favicon and Apple Touch icons are included in the root.
